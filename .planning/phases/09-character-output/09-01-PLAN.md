**Change 1 — Resonance Q boost for OB-X character (FILT-08):**
Before the Stage 1 `setParams()` call, compute a boosted resonance value for 24dB mode:
```cpp
float resonance24 = rack::clamp(resonance * 1.15f, 0.f, 0.95f);
```
Use `resonance24` (instead of raw `resonance`) for Stage 1: `filters[c].setParams(cutoffHz, resonance24, args.sampleRate)`.
For Stage 2, compute Q from the boosted value: `float q2 = resonance24 * 0.65f;` (was `resonance * 0.7f`). This produces sharper, more aggressive resonance peaks that cascade through both stages — mimicking the OTA ladder's peaky resonance characteristic vs the SEM's smooth plateau.

**Change 2 — Inter-stage saturation for harmonic character (FILT-08):**
Replace the simple clamp pass-through between stages with mild tanh saturation that adds odd harmonics proportional to signal level (like the CEM3320 diode-pair nonlinearity). Replace:
```cpp
interStage = rack::clamp(s1.lowpass, -12.f, 12.f);
```
With:
```cpp
interStage = rack::clamp(std::tanh(s1.lowpass * 0.12f) / 0.12f, -12.f, 12.f);
```
The 0.12f factor provides mild saturation: nearly linear for small signals (clean at low levels), increasing odd-harmonic content at higher signal levels and resonance (level-dependent character, authentic analog behavior). The `/0.12f` denominator normalizes so small signals pass at unity gain. The outer clamp maintains the existing safety boundary.

Note: The NaN/infinity check and filter reset MUST remain before this line (already exists, do not remove).

**Change 3 — HP/BP/Notch silence in 24dB mode (FILT-09 + TYPE-03):**
